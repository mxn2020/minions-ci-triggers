# Welcome to Minions Ci-triggers

We're excited to introduce **Minions Ci-triggers** — Code/prompt change listeners, CI/CD hooks, and regression gate definitions.

## What's Included

- **TypeScript SDK**: `npm install @minions-ci-triggers/sdk`
- **Python SDK**: `pip install minions-ci-triggers`
- **CLI**: `npm install -g @minions-ci-triggers/cli`

## Getting Started

Check out the [documentation](https://ci-triggers.minions.help) for a full guide.
