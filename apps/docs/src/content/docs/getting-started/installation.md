---
title: Installation
description: Install Minions Ci-triggers packages
---

## TypeScript / Node.js

```bash
npm install @minions-ci-triggers/sdk minions-sdk
```

## Python

```bash
pip install minions-ci-triggers
```

## CLI

```bash
npm install -g @minions-ci-triggers/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
