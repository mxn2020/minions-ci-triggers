# Minions Ci-triggers

Code/prompt change listeners, CI/CD hooks, and regression gate definitions

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-ci-triggers/sdk`) — Core library for Node.js
- **Python SDK** (`minions-ci-triggers`) — Python bindings
- **CLI Tool** (`@minions-ci-triggers/cli`) — Command-line interface
- **Web App** — Interactive playground at [ci-triggers.minions.wtf](https://ci-triggers.minions.wtf)
