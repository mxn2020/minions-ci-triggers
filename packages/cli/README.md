# @minions-ci-triggers/cli

CLI for Minions Ci-triggers — Code/prompt change listeners, CI/CD hooks, and regression gate definitions

## Installation

```bash
npm install -g @minions-ci-triggers/cli
```

## Usage

```bash
ci-triggers --help
```
